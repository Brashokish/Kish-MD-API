Kish-MD API

Music, Video Downloader & AI Chat powered by Gemini & Maher Zubair APIs

Available API Endpoints

/play?text=songname

Download MP3 audio from YouTube by song name.

POST /gpt

Chat with Gemini AI. JSON body: { "prompt": "your question" }

/tiktok?url=video_url

Download TikTok video metadata via Maher API.

/facebook?url=video_url

Download Facebook video metadata via Maher API.

/youtube?url=video_url

Download YouTube video metadata via Maher API.

/lyrics?text=songname

Get song lyrics via Maher API.
